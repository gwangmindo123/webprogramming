# The Beatles Fan Page

This is a simple fan page dedicated to the legendary band, The Beatles.

## Project Description

This project was created as a draft website to introduce The Beatles, including their history, members, and a basic photo gallery. The design is inspired by a blue color scheme.

## Files

- `index.html`: The main structure of the website.
- `style.css`: Contains the styling for the website, featuring a blue theme.
- `script.js`: A placeholder for future interactive elements.

## How to View

To view the website, simply open the `index.html` file in your web browser.
